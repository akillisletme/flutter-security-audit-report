# Mobile Security Report / Mobil Güvenlik Raporu

## English

This project contains a security report for Flutter, Android and iOS mobile applications. The report is intended to be used as a reference guide when reviewing and hardening the security of mobile apps. The same report is available in four languages under `reports/`.

- Report: [reports/security_en.md](reports/security_en.md)

## Türkçe

Bu proje, Flutter, Android ve iOS mobil uygulamaları için hazırlanmış bir güvenlik raporu içerir. Rapor, mobil uygulamaların güvenliğini incelerken ve sıkılaştırırken referans rehber olarak kullanılmak üzere hazırlanmıştır. Aynı rapor `reports/` altında dört dilde mevcuttur.

- Rapor: [reports/security_tr.md](reports/security_tr.md)

## Русский

Этот проект содержит отчёт по безопасности для мобильных приложений Flutter, Android и iOS. Отчёт предназначен для использования в качестве справочного руководства при проверке и усилении безопасности мобильных приложений. Тот же отчёт доступен на четырёх языках в папке `reports/`.

- Отчёт: [reports/security_ru.md](reports/security_ru.md)

## हिन्दी

इस प्रोजेक्ट में Flutter, Android और iOS मोबाइल ऐप्लिकेशन के लिए एक सुरक्षा रिपोर्ट शामिल है। यह रिपोर्ट मोबाइल ऐप्स की सुरक्षा की समीक्षा और उसे मज़बूत करने के दौरान संदर्भ मार्गदर्शिका के रूप में उपयोग के लिए तैयार की गई है। यही रिपोर्ट `reports/` में चार भाषाओं में उपलब्ध है।

- रिपोर्ट: [reports/security_hi.md](reports/security_hi.md)
